# HomeCritters — assets

Public image assets (screenshots, logos) for the
[HomeCritters](https://github.com/HomeCritters) project, so the org profile
and the Home Assistant plugin READMEs can embed them while the firmware repo
stays private.

## screenshots/

Captured from the real device (Spotpear Ball V2) and the web portal.

| File | Shows |
|---|---|
| `pet-day.png` | The pet in the forest (day theme) |
| `night-clock.png` | Night theme + idle NTP clock |
| `config-menu.png` | On-device config menu |
| `games-menu.png` | Games menu |
| `game-jump.png` / `game-fetch.png` / `game-genius.png` | The three mini-games |
| `party-mode.png` | Disco party mode (music playing) |
| `voice-ring.png` | Alexa-style voice ring (TTS) |
| `web-portal.png` / `web-portal-game.png` | React web portal + game controller |
